# Advisor AI
Advisor Portfolio Copilot is an AI-powered assistant that analyses client portfolios, identifies risk and performance gaps, and recommends compliant fund-level rebalancing. It evaluates asset allocation, portfolio drift, fund performance, and risk alignment, while incorporating recent fund-related news. Using an agentic approach, it flags only actionable opportunities with clear, advisor-friendly explanations and defensible rationale.

# To run
```
pip install -r requirements.txt
export GROQ_API_KEY="your_key"
streamlit run app.py
```
